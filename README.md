# Bible Runner

Bible Runner is plugin for KRunner that allows you to quickly open verse in YouVersion Bible.

## Installation

Install dependencies:

```bash
apt install libkf5kio-dev ibkf5notifications-dev cmake make x-www-browser
```

Run install script:

```bash
./install.sh
```
